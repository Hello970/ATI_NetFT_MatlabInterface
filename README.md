# ATI_NetFT_MatlabInterface
Basic code to sample the the ATI Net-FT sensors from within MATLAB

# Setup
* TCP/IPv4 settings: static IP 192.168.1.100, submask 255:255:255:0
* sensor-IP: 192.168.1.1
